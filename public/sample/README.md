# Portfolio Layout Samples

This directory contains 15 unique portfolio layout designs, all using the same color scheme optimized for FAANG/master's degree applications.

## Color Scheme

All layouts use a consistent, professional color palette:
- **Accent Teal**: `#83d4c5` - Primary accent color
- **Light Sky Blue**: `hsla(190, 52%, 67%, 1)` - Headings and highlights
- **Gainsboro**: `hsla(90, 8%, 85%, 1)` - Body text
- **Dark Background**: `#302f2f` - Card backgrounds
- **Black**: `#010101` - Main background

## Customization Tips

1. **Content**: Replace placeholder text with your actual experience
2. **Images**: Add project screenshots or professional photos
3. **Links**: Update social media and contact links
4. **Colors**: While the color scheme is consistent, you can adjust accent colors
5. **Animations**: Reduce or remove animations for more conservative applications
