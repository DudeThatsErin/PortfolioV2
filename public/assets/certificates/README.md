# Certificates Directory

This directory is for storing your certificate images that will be automatically displayed on your portfolio's resume page.

## How to Use

1. Upload your certificate images (PNG, JPG, JPEG, GIF, or WEBP) to this directory
2. The images will automatically appear on your resume page at erinskidds.com/resume
3. Click on any certificate to view it in a full-screen lightbox
4. Use arrow keys or navigation buttons to browse through multiple certificates

## Supported Formats

- PNG (recommended for certificates)
- JPG/JPEG
- GIF
- WEBP

## File Naming

The filename (without extension) will be used as the certificate title. For example:
- `udemy-react-course.png` → displays as "Udemy React Course"
- `AWS_Certified_Developer.jpg` → displays as "AWS Certified Developer"

Underscores and hyphens in filenames are automatically converted to spaces.

## Features

- **Automatic Detection**: Just drop files here - no code changes needed
- **Responsive Grid**: Adapts to all screen sizes
- **Lightbox Viewer**: Click to view full-size with navigation
- **Keyboard Navigation**: Use arrow keys (←/→) and Escape to navigate
- **Mobile Optimized**: Touch-friendly and responsive
